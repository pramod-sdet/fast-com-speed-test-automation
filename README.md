# 🚀 Fast.com Speed Test Automation

This project automates the internet speed test on **Fast.com** using **Java + Selenium WebDriver**.

The script continuously reads the changing speed values during the test and prints the **final internet speed result** once the test completes.

---

## 🧪 What This Automation Does

- Opens **https://fast.com**
- Starts the speed test automatically
- Continuously captures the changing speed values
- Detects when the test finishes
- Prints the **final speed result**

---

## 🛠 Tech Stack

- Java
- Selenium WebDriver
- Maven
- ChromeDriver

# 🚀 SpeedCheckAutomation  

This project is a **Java + Selenium automation script** that checks internet speed on [Fast.com](https://fast.com/).  

The script **continuously monitors and prints the changing speed values and units** (like Mbps) in real time, and finally prints the **final speed result** after the test completes.    

---

## 📌 What It Does  

- Opens **Fast.com** in the browser  
- Continuously fetches and prints the **real-time speed values**  
- Detects when the test stops  
- Prints the **final internet speed result**  

---

## 🛠️ Tech Used  

- **Java**  
- **Selenium WebDriver**
